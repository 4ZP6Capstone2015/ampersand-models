== GEMEENTE BALIE SCRIPT ==
Iemand komt bij de gemeentebalie met een reeds ingevuld, papieren VOG aanvraagformulier. Deze moet dan door een gemeenteambtenaar elektronisch naar COVOG worden gestuurd. Volgens de wet kan dat echter alleen als de persoon is ingeschreven in die gemeente. Volgens de wet moet de gemeente instaan voor de volledigheid van de aanvraag alsmede voor de identiteit van de aanvrager.

- Stap 1: Opstarten voor de rol van Gemeente
+ Start de VOG Demo applicatie, 
+ reset naar de default populatie 
+ selecteer de rol 'Gemeente'
+ Klik op 'G: Overzicht'

-> Stap 2: Ambtenaar logt in
+ Merk op dat op het scherm ook staat dat de ambtenaar moet inloggen met zijn GemeenteID
+ Selecteer 'Login' op de menubalk
+ klik op 'Edit'
+ Vul de velden als volgt in:
  userid = noordenveld
  password = *****
(als je dit verkeerd doet komt er geen foutmelding, dat is een bug in de prototypegenerator die nog gemeld moet worden)
+ klik op 'Save'
+ klik op 'G: Overzicht' in de menu balk
(het overzicht dat wordt getoond is voor ambtenaar N. Oordenveld)

-> Stap 3: Ambtenaar maakt nieuwe aanvraag
+ Klik op 'New' in de menubalk en selecteer 'GemeentelijkeVOGZaak (Ga: Nieuwe VOG Aanvraag)'
+ vul de velden als volgt in:
  aanvraagnummer = 123456
  Gemeenteid = noordenveld
  wachtwoord = *****
+ Klik op 'Save'
(er verschijnt een signaal dat zegt dat een vog aanvraag ingevuld moet worden)

-> Stap 4: Ambtenaar vult aanvraag in - deel 1
+ Klik op de zaak in het signaal
+ selecteer de actie 'GemeentelijkeVOGZaak (Ga: Nieuwe VOG Aanvraag)'
+ klik op 'Edit' 
  + klik op (insert new) 'VOG Aanvraag' en vul de velden als volgt in:
    Doel van de aanvraag = Acme B.V. : Financieel medewerker
    Soort identiteitsbewijs (paspoort, rijbewijs) = Paspoort
    Serienummer = P453027177
  + klik op 'Save'
(er verchijnt een foutmelding: De aanvrager is niet ingeschreven in deze gemeente, en het wetsartikel waarin staat dat dit niet mag)
(De persoon aan de palie wordt doorverwezen naar zijn eigen gemeente, de gemeente Veere.)

(Nu verschijnt iemand anders aan de balie, ook met een vooringevuld, papieren aanvraagformulier:) 
-> Stap 5: Ambtenaar vult aanvraag in - deel 2
+ verwijder (klik op het kruisje voor het betreffende veld):
  + Soort identiteitsbewijs (paspoort, rijbewijs)
  + Serienummer
+ vul de volgende velden als volgt in
  + DigiD = kljansen
  + Wachtwoord = *****
+ klik op 'Save'
(nu mag het wel, wat meneer Jansen is ingeschreven in Noordenveld)

-> Stap 6: Ambtenaar vult aanvraag in - deel 3
+ vul de volgende velden als volgt in (klik op 'Edit')
  Bijzonderheden geconstateerd (Ja/Nee)? = 'Nee' 
  Persisteren in de aanvraag (Ja/Nee)? = 'Nee' 
  COVOG om advies gevraagd omtrent ontvankelijkheid (Ja/Nee)? = 'Nee' 
+ zorg dat het gele 'overtredingen'veld (rechtsonder) groot is
  (als het goed is staat er een melding dat de aanvraag nog gecompleteerd moet worden)
+ klik op 'Save'
  (de melding verandert nu. Er staat dat de aanvraag ondertekend moet worden)

-> Stap 7: Ambtenaar ondertekt de aanvraag
+ klik op 'Edit' 
+ vul de volgende velden als volgt in
  datum = [vul de datum van vandaag in]
  handtekening = N. Oordenveld
+ klik op 'Save'
  (het scherm wordt zo goed als leeg - de aanvraag is naar het COVOG gestuurd)

-> Stap 8: Ambtenaar kan opgestuurde zaken volgen
+ klik op 'G: Overzicht' in de menu balk
  (je ziet dat de vog aanvraag in behandeling is)
+ klik op het pijltje (->) in de box 'uitstaande aanvragen'
  ++ en selecteer 'G2: Opgestuurde VOG aanvragen'
  (je ziet nu een uittreksel van het invulformulier; je mag niets meer veranderen)
