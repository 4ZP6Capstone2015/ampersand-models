Prototyping van user story "Urentegoeden" voor HR, in het kader van NVWA DTV
Benodigdheden: 1 browsersessie (Firefox c.q. Chrome) op internet

open de applicatie "http://sentinel.tarski.nl/ampersand/NVWA/"  (login en password nodig)

Validatie van dit prototype door deskundigen:


Stap 0: (indien gewenst) Initialiseer het prototype door de database te resetten:
+ in de browser: http://sentinel.tarski.nl/ampersand/NVWA/Installer.php
--> Stel vast dat de browser "OK" terugmeldt. Incidenteel kan dit enige tijd (<30sec) vergen.
+ in de browser: http://sentinel.tarski.nl/ampersand/NVWA/


Stap 1:
+ Klik rechts bovenin de rol van "HR" aan


Stap 2
+ Klik op "Overzicht urentegoeden" in het hoofdmenu (de zwarte balk)
--> Stel vast dat een lijst van medewerkers met hun urentegoeden zichtbaar is.
+ Druk op het pijltje in de bovenrand van medewerker 22222222.
+ Kies de interface "Wijzig urentegoeden"
+ Druk op de knop Edit
+ Maak een nieuw urentegoed aan
--> Stel vast dat één van de beschikbare urenpools kan worden geselecteerd.
--> Stel vast een aantal uren kan worden ingevuld.
--> Stel vast dat een periode kan worden gekozen.
+ Probeer één van de velden 'urenpool', 'saldo', of 'tot en met' leeg te laten en druk op SAVE
--> Rapporteer een fout wanneer dit lukt.
--> Stel vast dat dit verplichte velden zijn.
+ Probeer als urenpool "Aftopuren" in te vullen.
--> Rapporteer een fout wanneer dit lukt.

Stap 3
We gaan nu een nieuwe urenpool maken. Daarvoor moeten we de rol van FAB aannemen.
+ Klik rechts bovenin de rol van "FAB" aan
+ Klik op "Beheer Urenpools" in het hoofdmenu (de zwarte balk)
+ Probeer de urenpool "Overwerkverlof" te veranderen in "Overwerk"
--> Rapporteer een fout wanneer dit lukt.
+ Voeg een urenpool "Aftopuren" toe en klik op Save.
--> Stel vast dat dit lukt.

Stap 4
We gaan nu alsnog aftop-uren invoeren.
+ Klik op "Overzicht urentegoeden" in het hoofdmenu (de zwarte balk)
+ Druk op het pijltje in de bovenrand van medewerker 22222222.
+ Kies de interface "Wijzig urentegoeden"
+ Druk op de knop Edit
+ Maak 83 aftop-uren aan voor medewerker 22222222, in de meest recente periode.
--> Stel vast dat het nu wél lukt om aftop-uren te registreren.
