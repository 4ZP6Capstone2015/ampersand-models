Prototyping van user story "Urentegoeden" voor P&O, in het kader van NVWA DTV
Benodigdheden: 1 browsersessie (Firefox c.q. Chrome) op internet

open de applicatie "http://sentinel.tarski.nl/ampersand/NVWA/"  (login en password nodig)

Validatie van dit prototype door deskundigen:


Stap 0: (indien gewenst) Initialiseer het prototype door de database te resetten:
+ in de browser: http://sentinel.tarski.nl/ampersand/NVWA/Installer.php
--> Stel vast dat de browser "OK" terugmeldt. Incidenteel kan dit enige tijd (<30sec) vergen.
+ in de browser: http://sentinel.tarski.nl/ampersand/NVWA/


Stap 1:
+ Klik rechts bovenin de rol van "FAB" aan


Stap 2
In deze stap gaan we P-Direkt codes toevoegen en wijzigen.
+ Klik op "Beheer P-Direkt codes" in het hoofdmenu (de zwarte balk)
--> Stel vast dat een lijst van P-Direkt codes zichtbaar is.
+ Klik op de knop Edit
+ Wijzig de code en de rubriek van de eerste P-Direktcode in de lijst (nl. 0235)
--> Stel vast dat je de code kunt wijzigen, ook met een waarde anders dan genoemd in de selectie.
--> Stel vast dat je de rubriek kunt wijzigen, ook met een waarde anders dan genoemd in de selectie.
--> Stel vast dat je de invoer kunt wijzigen, ook met een waarde anders dan genoemd in de selectie.
--> Stel vast dat je de opmerking kunt wijzigen.
Maak een nieuwe P-Direktcode aan:
+Hover met de muis over het eind van de lijst, totdat "Insert new P-Direkt code" zichtbaar wordt. Klik daarop.
--> vul de velden in.
--> Stel vast dat je een nieuwe P-Direkt code kunt invullen.
+ Probeer een P-Direkt code te maken waarvan de code al in gebruik is.
--> Rapporteer een fout wanneer dit lukt.

Stap 3
In deze stap gaan we toelages wijzigen en een nieuwe toelage invoeren. Stel vast dat je de rol van FAB hebt (selecteer anders rechts bovenin de rol van "FAB")
+ Klik op "Beheer Toelagen en Toeslagen" in het hoofdmenu (de zwarte balk)
+ Probeer van het recht "Aflopende toelage" de soort "recht" te veranderen in "Overwerk"
--> Rapporteer een fout wanneer dit lukt.
+ Probeer het recht "Aflopende toelage" te verwijderen uit het systeem.
--> Rapporteer een fout wanneer dit lukt.
+ Probeer "salarisschaal14" te verwijderen uit het systeem.
--> Rapporteer een fout wanneer dit lukt.
+ Probeer "categorie3" te verwijderen uit het systeem.
--> Rapporteer een fout wanneer dit lukt.
+ Voeg een salarisschaal "salarisschaal14" toe aan deze referentielijst
--> Rapporteer een fout wanneer dit lukt.
+ Voeg een salarisschaal "salarisschaal19" toe aan deze referentielijst
--> Stel vast dat dit lukt.
