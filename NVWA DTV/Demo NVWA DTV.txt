- Demo NVWA DTV
Benodigdheden: 2 browsersessies (Firefox en Chrome)

-- Scherm 1: open de applicatie "localhost:8080/NVWA" in Firefox
-- Scherm 2: open de applicatie "localhost:8080/NVWA" in Chrome
De demo kan nu beginnen.

Stap 1 (Scherm 1 en 2):
+ Scherm 1: Klik rechts bovenin de rol van "Medewerker" aan
+ Scherm 2: Klik rechts bovenin de rol van "Leidinggevende" aan


Stap 2 (Scherm 1: Medewerker)
+ Klik op "Login" in het hoofdmenu
+ Vul de velden als volgt in:
  e-mail adres: benny.beer@nvwa.nl
  wachtwoord    : welkom
+ Klik op Save

Stap 3 (Scherm 2: Leidinggevende)
+ Klik op "Login" in het hoofdmenu
+ Vul de velden als volgt in:
  e-mail adres: stef.joosten@ou.nl
  wachtwoord    : welkom
+ Klik op Save

----- Aanvragen autorisatie door werknemer -----

Stap 4 (Scherm 1: Medewerker)
+ Klik op "Main" in het hoofdmenu
+ Klik op "Create new Autorisatie(Aanvragen Autorisatie)"
+ Vul de velden als volgt in:
  "voor medewerker" : Benny Beer
  "geldig van" : 01-01-2014
  "geldig tot" : 31-12-2014
  "project" : DTV
  "tijdCategorieen" : Werktijd
+ Klik op "Insert new TijdRubriekCategorie"
  "tijdCategorieen" : Pauze
  "tijdstip aanvraag" : sysdate
  "te verlenen door" : Stef Joosten
+ Klik op Save.

----- Bijhouden (toekennen en intrekken) van autorisaties door leidinggevende -------

Stap 5 (Scherm 2: Leidinggevende)
+ Klik op "Openstaande autorisaties" in het hoofdmenu
+ Klik op Edit
+ Vul bij status de waarde "Verleend" in
+ Klik op Save, de autorisatie is nu verleend

+ Klik op "Autorisaties beheren" in het hoofdmenu
+ Klik op Edit
+ De status (Ingetrokken of Verleend) van alle autorisaties kan nu worden aangepast
+ Klik op save

----- Werkrooster inregelen ------


Stap 6 (Scherm 1: Medewerker)    **** Hoort vanaf de leidinggevende te lopen. Wegens issue in generator nu via medewerker ****
+ Klik op "Werkrooster inregelen" in het hoofdmenu
+ Klik op Edit
+ Er kunnen voor de medewerker roostertijden worden aangepast, 
  roostersoorten en tijden worden toegevoegd op een dag (Insert new Dienst), 
  roosters worden aangemaakt voor een nieuwe dag (Insert new Dagverantwoording)
+ Klik op save

----- Invullen timesheet door medewerker ---------

Stap 7 (Scherm 1: Medewerker)
+ Klik op "Nieuwe Dagverantwoording" in het hoofdmenu
+ Klik op Edit
+ Vul bij de dagverantwoording met betreft datum "27-8-2014" de velden als volgt in:
  "datum invoer" : sysdate
+ Klik op "tijdverantwoording"
  "begin" : 8:00
  "eind"  : 12:00
  "tijdrubriek" : Inspectie
  "project" : DTV
+ Klik op "insert new Tijdverantwoording
  "begin" : 12:00
  "eind"  : 12:30
  "tijdrubriek" : Pauze
  "project" : DTV
+ Klik op "insert new Tijdverantwoording
  "begin" : 12:30
  "eind"  : 16:30
  "tijdrubriek" : Inspectie
  "project" : DTV
+ Klik op Save, zie foutmelding over geen autorisatie
+ Klik op Save, dagverantwoording is nu opgeslagen


----- Dagverantwoording retour -> wijzigen -> akkoord -------


Stap 8 (Scherm 2: Leidinggevende)
+ Klik op "Te valideren" in het hoofdmenu (op witte balk, dus alleen zichtbaar bij mouse-over)
+ Klik op Edit
+ Vul bij akkoord de status "Retour" in
+ Klik op save

stap 9 (scherm 1: Medewerker)
+ Klik op "Retour" in het hoofdmenu
+ Klik op Edit
+ Verander eindtijd van Pauze tijdverantwoording naar 13:00 en begintijd van 2e inspectie registratie naar 13:00
+ Klik op save

Stap 10 (Scherm 2: Leidinggevende)
+ Klik op "Te valideren" in het hoofdmenu (op witte balk, dus alleen zichtbaar bij mouse-over)
+ Klik op Edit
+ Verander de waarde van de status bij het veld akkoord naar "Akkoord"
+ Klik op save

Stap 11 (Scherm 2: Medewerker)
+ Klik op "Geaccordeerde tijdverantwoordingen" in het hoofdmenu
+ Zie de geaccordeerde dagverantwoording


----- Berekenen van toeslaguren door expert -------

Stap 12 (Scherm 1: Medewerker)
+ Klik rechts bovenin de rol van "RuleEngine" aan
+ Klik op "Rule engine" in het hoofdmenu
+ Zie de ingevulde dagverantwoording en bereken indien mogelijk toeslaguren
+ Klik op Edit
+ Vul bij het veld "Resultaat verwerking" eventueel het resultaat van de berekeningen in
+ Klik op Save
