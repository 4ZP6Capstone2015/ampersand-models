README file for Ampersand app CP23

Revision history:
RJ/20140527: created the first version.

APPLICATION CONTENT:
The purpose of this app is to show what the result can be of interpreting a policy document, which in this example is a policy document of the fictitious Company Inc, and deals with portable equipment that may be issued to employees. The policy is in the directory as a DOC, DOCX or PDF document.  

APPLICATION IMPLEMENTATION SPECIFICS:
The application makes use of ExecEngine rules to compute the staus of employees (see "CP23 Status.adl"). Rudimentary box coloring is a relatively recent addition that you can find in 'index.php'.

GETTING IT TO WORK:
execute 'prototype.exe CP23.adl', run the installer and off you go.
