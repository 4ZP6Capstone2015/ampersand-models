EBV Document Header

Deze gegevenselementen komen aan het begin van elke xsd te staan. Door de manier van implementeren is er echter geen sprake van een echte gegevensgroep Bedrijfsdocument. Het wordt ook niet apart gespecificeerd bij elk bedrijfsdocument, maar staat impliciet altijd aan het begin van een xsd.

Property term Cardinality Definition
ID 1..1 Unieke identificatie van een bedrijfsdocument.
VerzendendeInstantie 1..1 De instantie die het bedrijfsdocument heeft verzonden.
Instantiecode 1..1 Code die een instantie of een onderdeel daarvan identificeert.
OntvangendeInstantie 1..1 De instantie voor wie het bedrijfsdocument is bestemd.
Instantiecode 1..1 Code die een instantie of een onderdeel daarvan identificeert.
Datum 1..1 De datum waarop het bedrijfsdocument is aangemaakt.
Tijdstip 1..1 Het tijdstip waarop het bedrijfsdocument is aangemaakt.
TestIndicator 1..1 Aanduiding of het bedrijfsdocument voor testdoeleinden is bedoeld.
SchemaVersieID 0..1 Identificatie van de versie (major/minor) van het XML Schema waaraan het document voldoet.
