- Compileer DemoZaakgegevens.adl
- Demo "Straf zaaksproces"
Benodigdheden: vier beeldschermen, die toegang hebben tot dezelfde database

-- Scherm 1: Open de applicatie "localhost/strafzaak" in Firefox
-- Scherm 2: Open de applicatie "localhost/strafzaak" in Firefox
-- Scherm 3: Open de applicatie "localhost/strafzaak" in Firefox
-- Scherm 4: Open de applicatie "localhost/strafzaak" in Firefox
-- Scherm 5: Open de applicatie "localhost/strafzaak" in Firefox
De demo kan nu beginnen


=====================
== Aanbrengen zaak ==
=====================

Stap 1 (Scherm 1, 2, 3 en 4):
+ Scherm 1: Klik rechts bovenin de rol van "Officier van Justitie" aan
+ Scherm 2: Klik rechts bovenin de rol van "Administratief medewerker" aan
+ Scherm 3: Klik rechts bovenin de rol van "Juridisch medewerker" aan
+ Scherm 4: Klik rechts bovenin de rol van "Poortraadsheer" aan


Stap 2 (Scherm 1: Officier van Justitie):
+ Klik op "Appointeringsvoorstel indienen" en vul de velden als volgt in:
  Kenmerk                  : App322
  Parketnummer(s)          : 12/0223
  Klik op "Insert nieuw Zaakid" om meerdere zaken toe te voegen
  Parketnummer(s)          : 12/0328
  Roldatum                 : 22-07-2012

+ Klik op "Save"


Stap 3 (Scherm 4: Poortraadsheer):
+ Klik op "Zaken" in de zwarte menubalk
+ Klik bij "Appointeer zaken" op zaak 12/0123 en vul de velden als volgt in
  Gegegevens zitting
  Geschatte behandeltijd : 40 min
  Behandeldatum          : 22-07-2012
  Informatie over kamer
  Samenstelling          : Enkelvoudig
  Naam kamer             : EK 1
+ Klik op "Save"


============================
== Behandeling op zitting ==
============================

Stap 5 (Scherm 2: Administratief medewerker)
+ Registreer een zitting, door "Een nieuwe zitting plannen" aan te klikken
+ Vul de velden als volgt in:
  Zaal            : A
  Zaken
  Behandeling van : 12/0123
  Klik op "Insert nieuwe behandeling" om meerdere zaken toe te voegen"
  Behandeling van : 11/0863
  Bezetting
  Rechter(s)      : mr. H.J. Deuring
  Griffier        : G.M. Fondse
  Datum           : 22-07-2012
  Aanvangstijdstip: 11:00
  Eindtijd        : 12:30

+ Klik op "Save"


Stap 6 (Scherm 3: Juridisch medewerker)
+ De Juridisch medewerker maakt het pv op. Klik op Proces-verbaal in de zwarte menubalk
+ Klik bij "Proces-verbaal opmaken voor" op "De behandeling van 12/0123"
+ Vul de velden als volgt in:
  Klik op Proces-verbaal
  Type document           : Proces-verbaal
  Kenmerk                 : pv 12/0123-1
  Beslissingen            : De rechter doet uitspraak over 2 weken
  Griffier                : G.M. Fondse
  Behandeling gesloten om : 10:40
  

Stap 7 (Scherm 2: Administratief medewerker)
+ Als de behandeling gesloten is kan de griffie de zitting afronden door het proces-verbaal te registreren
+ Klik op Zittingen in de zwarte menubalk
+ Klik op "12/0123" onder "Af te ronden behandelingen" (gaat naar "Behandeling afronden")
+ Klik op "Edit"
+ Vul bij het Proces-verbaal "pv 12/0123-01" in en klik op "save"


====================
== Vonnis opmaken ==
====================
Stap 8 (Scherm 3: Juridisch medewerker)
+ Klik op "Vonnissen", hier staan de zaken waarvoor nog een vonnis geschreven moet worden
+ Klik op 12/0123
+ Klik op "Vonnis schrijven" 
+ Klik op "Edit" en vul de velden als volgt in
  Kenmerk         : Von12/0123
  Type beslissing : Eindvonnis
  Beslissing      : Schuldig bevonden  
+ Klik op "Save"


Stap 9 (Scherm 5: Rechter)
+ Log in als rechter door in de zwarte menubalk op "Login" te klikken
+ Klik op "edit" en vul de velden als volgt in:
  Gebruikersnaam : mr. H.J. Deuring
  Wachtwoord : de
+ Klik op "save"
+ Klik op "Mijn pagina" en op zaaknummer 12/0123
+ Klik op "Vonnis schrijven" om te zien dat de rechter het vonnis van de juridisch medewerker kan bewerken
+ Klik op "Vonnis" in de bovenste zwarte menubalk
+ Bij "Te ondertekenen vonnissen" klik op 12/0123
+ Klik op "Vonnis ondertekenen"
+ Vul de datum in "16-05-2012" en de naam van de rechter, "mr. H.J. Deuring" in en klik op "save"
